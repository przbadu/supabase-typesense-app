# Supabase Typesense App

This is a starter project for a full-stack app using Supabase and Typesense.

## Setting up

1. Copy the `.env.example` file to `.env` and fill in the environment variables.

```sh
cp .env.example .env
```

2. Setup the project

```sh
bin/setup
```

3. Run the app

```sh
bin/dev
```
